Help — how to fix Gujarati / Hindi / Marathi text (nothing to install)

You only need a web browser and the downloaded zip. No fonts or special software required.

1. Open your files

  1. Download a book zip from the table above and unzip it.
  2. Open the pages folder — open a page image (e.g. page_004.png) and keep it visible.
  3. Open ocr-draft.txt (double-click on computer; on phone use Notes or any text app). Wrapped lines are already joined into paragraphs — blank lines mark new paragraphs.

2. Type Gujarati, Hindi, or Marathi online (free)

  1. Open Google Input Tools in your browser.
  2. Choose your language: Gujarati (ગુજરાતી), Hindi (हिन्दी), or Marathi (मराठी).
  3. Click in the text box and type the word in English letters as you say it — e.g. pavitra, gaayatri.
  4. When the correct script appears, select it → Copy.
  5. Go back to ocr-draft.txt, find the wrong word, delete it, and Paste.
  6. Check against the page image — the text should match the picture.

Gujarati menu version (same tool): google.com/intl/gu/inputtools/try

3. Save and upload

  1. Keep every page marker exactly as-is: ===== PAGE 1 =====
  2. When finished, save as ocr-corrected.txt.
  3. On this page, under Submit corrected, enter your name, choose the file, and click Upload.

Optional: Google Docs

  1. Go to docs.google.com (free Google account).
  2. Paste all of ocr-draft.txt into a new document.
  3. Use Google Input Tools (step 2) in another tab to copy corrected words into the document.
  4. File → Download → Plain text (.txt), rename to ocr-corrected.txt.
